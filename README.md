# Activ Chatbot – Privacy Policy & Docs

This repository hosts the public Privacy Policy URL required for Facebook App Review.

- Privacy Policy (VI): ./privacy-policy.md
- Direct link you can submit to Facebook (after pushing to GitHub):
  - Pretty view: https://github.com/<your-account>/<your-repo>/blob/main/privacy-policy.md
  - Raw URL: https://raw.githubusercontent.com/<your-account>/<your-repo>/main/privacy-policy.md

Replace contact details in `privacy-policy.md` with your real company info before submitting.

## How to update
- Edit `privacy-policy.md`
- Commit & push to update the public URL.
